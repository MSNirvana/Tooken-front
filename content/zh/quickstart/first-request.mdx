## 最小调用示例

```bash
curl https://api.tooken.ai/v1/chat/completions \
  -H "Authorization: Bearer $TOOKEN_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"model":"auto","messages":[{"role":"user","content":"Hello"}]}'
```
