## 端点

`POST /chat/completions`

## 说明

可传入固定模型或 `auto` 路由策略。

## 请求参数

<ParamTable>
  <Param name="model" type="string" required>
    模型 ID，或 `auto / auto-fast / auto-cheap / auto-quality`
  </Param>
  <Param name="messages" type="array" required>
    对话消息列表，至少包含一条用户消息
  </Param>
  <Param name="stream" type="boolean">
    是否启用流式响应，默认 `false`
  </Param>
</ParamTable>
