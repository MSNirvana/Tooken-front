## 平台定位

Tooken 提供统一的 `base_url` 与 API 协议，让你在不改业务结构的前提下接入多模型能力。

## 核心价值

- 一个密钥，调用多个模型供应商
- 支持自动路由策略，按成本与效果动态选择模型
- 提供企业级配额、审计与支付能力

<Callout type="success">如果你之前已接入 OpenAI SDK，通常只需替换 `base_url` 即可迁移到 Tooken。</Callout>

## 最小示例

```bash {2,5-6}
curl https://api.tooken.ai/v1/chat/completions \
  -H "Authorization: Bearer $TOOKEN_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "auto",
    "messages": [{"role": "user", "content": "Hello"}]
  }'
```
